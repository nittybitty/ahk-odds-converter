# Odds Calculator (AutoHotkey)

A simple AutoHotkey GUI tool for converting between different odds formats used in betting and prediction markets.

## Features

- **Decimal to American Odds** - Convert decimal odds to American format with implied probability
- **Prediction Market Percentages** - Convert yes/no prediction market probabilities to American odds
- **American to Decimal** - Convert American odds back to decimal format with implied probability

## Usage

1. Download [odds-calculator.ahk](odds-calculator.ahk)
2. Install [AutoHotkey](https://www.autohotkey.com/) if you haven't already
3. Double-click the `.ahk` file to run

Or compile it to a standalone `.exe` using AutoHotkey's Ahk2Exe compiler.

## Examples

- **Decimal 2.5** → American: +150 | Implied: 40%
- **Prediction Market 65%** → American: -186 | Decimal: 1.54
- **American +200** → Decimal: 3.00 | Implied Prob: 33.33%
- **American -150** → Decimal: 1.67 | Implied Prob: 60%
